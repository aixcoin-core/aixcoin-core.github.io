--- name: generate aixversion: 0.18.0 aixgroup: generating permalink: en/doc/0.18.0/rpc/generating/generate/ --- generate nblocks ( maxtries ) Mine up to nblocks blocks immediately (before the RPC call returns) to an address in the wallet. Arguments: 1. nblocks (numeric, required) How many blocks are generated immediately. 2. maxtries (numeric, optional, default=1000000) How many iterations to try. Result: [ blockhashes ] (array) hashes of blocks generated Examples: Generate 11 blocks > aixcoin-cli generate 11
